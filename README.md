# 仓储运输管理系统

《仓储运输管理系统》是一款专门针对企业仓库或配送中心而设计的实时仓储作业管理系统。系统包含： 1、基本信息：公司信息、客户信息、车辆信息、货品信息 2、仓库管理：入仓登记、出仓登记 3、统计汇总：入仓明细、出仓明细、客户余量 




# 如何运行本系统?

本系统基于宏达数据库信息管理开发平台开发，下载后，双击文件夹中"dbimp.exe(或Hadp.exe)"文件即可完整运行对应的信息管理系统。
